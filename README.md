# 🔮 Python Trivia Game

A spooky terminal-based trivia game built with Python covering
Supernatural, Spirit Realm, Tarot, and Machine Learning topics!

## 🚀 How to Run

Make sure you have Python installed, then run:
python3 Python_Trivia_Game.py

## 🎮 How to Play

- You will be asked 5 random questions
- Type your answer and press Enter
- Answers are not case sensitive (vampire = Vampire ✅)
- Your score will be shown at the end

## 📚 Topics Covered

- 👻 Spirit Realm
- 🔮 Tarot
- 🧛 Supernatural Entities
- 🤖 Machine Learning & AI

## 🛠️ Built With

- Python 3
- random module

## 💡 Features

- Random 5 questions picked every time you play
- Case insensitive answers
- Score tracker
- Two completely different topic categories — spooky & tech!
